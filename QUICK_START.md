# 🚀 Quick Start Guide

Get up and running in 5 minutes!

## Step 1: Install Dependencies (1 min)

```bash
cd stellar-wallet-checker
npm install
```

Wait for installation to complete...

## Step 2: Install Freighter Wallet (2 min)

1. Visit: https://www.freighter.app/
2. Click "Download" for your browser
3. Install the extension
4. Set up a new wallet (save your recovery phrase!)
5. Open Freighter → Settings → Select "TESTNET"

## Step 3: Fund Your Testnet Account (1 min)

1. Copy your public key from Freighter (click to copy)
2. Visit: https://laboratory.stellar.org/#account-creator?network=test
3. Paste your public key
4. Click "Get test network lumens"
5. Wait for confirmation (you'll get 10,000 test XLM)

## Step 4: Run the Application (1 min)

```bash
npm run dev
```

Open browser to: http://localhost:3000

## Step 5: Test the Application

1. Click "Connect Freighter Wallet"
2. Approve the connection
3. See your balance load
4. Try sending a small transaction (1 XLM to yourself)
5. View the success message!

## 🐙 Push to GitHub

### Quick Method (Using GitHub Desktop):
1. Download: https://desktop.github.com/
2. Add repository → Choose the stellar-wallet-checker folder
3. Publish repository → Make it public
4. Done!

### Command Line Method:
```bash
# In the stellar-wallet-checker folder
git init
git add .
git commit -m "Initial commit: Stellar Wallet Balance Checker"

# Create repo on GitHub first, then:
git remote add origin https://github.com/YOUR-USERNAME/stellar-wallet-checker.git
git branch -M main
git push -u origin main
```

## 🆘 Need Help?

- **Application won't run**: Delete `node_modules` folder, run `npm install` again
- **Freighter not detected**: Refresh page after installing Freighter
- **Balance shows 0**: Make sure you funded your testnet account
- **Git issues**: Use GitHub Desktop (easiest option)
